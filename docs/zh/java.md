# Java （Android 项目）

flow.ci 当前支持以下 Java （Android 项目）版本：

- JDK     (jdk7、jdk8)
- SDK  （android -14 ~ android - 23 ）
 
## Gradle

Android 项目使用 Gradel 作为构建工具
