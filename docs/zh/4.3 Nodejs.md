## 4.3 Node.js

### flow.ci 当前支持以下 Node.js 版本：

- Node.js v0.10.32
- Node.js  v0.12.2
- Node.js  v4.0.0
- Node.js  v5.2.0

### 依赖安装

Node.js 项目默认使用 NPM 作为依赖管理工具
